## 📊 实验结果

### 实验1: 增益扫描
✅ 成功绘制不同背景光强下的增益-灰度响应曲线
- 低光照(20): 最大灰度 25.52 @ 11.84dB
- 正常光照(50): 最大灰度 54.75 @ 17.14dB
- 高光照(100): 最大灰度 103.48 @ 2.04dB
- 很高光照(150): 最大灰度 152.20 @ 8.16dB

### 实验2: 算法比较
✅ 对比基础算法和自适应算法性能
- 基础算法平均得分: 65.67
- 自适应算法平均得分: 65.33
- 基础算法平均迭代: 2.27次
- 自适应算法平均迭代: 2.91次

**结论**: 两种算法性能接近,基础算法收敛略快

### 实验3: 噪声鲁棒性
✅ 验证算法在不同噪声水平下的稳定性
- 噪声范围: 0.5 - 10.0
- 平均灰度误差: 200.25 ± 0.00
- 优化得分: 60.13 (稳定)

**结论**: 算法对噪声具有良好的鲁棒性

### 实验4: 收敛性分析
✅ 从11个不同初始增益测试收敛特性
- 所有初始条件都能收敛
- 平均迭代2-3次
- 初始增益对最终结果影响小

**结论**: 算法收敛快速且稳定
